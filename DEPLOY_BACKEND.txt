AGGIORNAMENTO BACKEND (Apps Script) - dDAE Drive per-utente

Se in Impostazioni compare l'errore "Action non valida: drive_root",
significa che la Web App Apps Script sta eseguendo una versione vecchia.

Procedura corretta:
1) Apri il tuo progetto Google Apps Script collegato al Google Sheet.
2) Apri il file Code.gs e sostituisci TUTTO il contenuto con il Code.gs incluso in questo ZIP.
3) Deploy → Manage deployments → (seleziona la Web App) → Edit
4) Seleziona "New version" e poi Deploy.
5) Verifica che la Web App sia impostata su:
   - Execute as: User accessing the web app
   - Who has access: Anyone with the link (o la tua policy prevista)
6) Se l'URL della Web App cambia, aggiorna API_BASE_URL in config.js e ridistribuisci la PWA su GitHub.

Nota:
Ogni utente deve autorizzare Drive la prima volta che usa le funzioni Drive.
